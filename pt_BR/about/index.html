About

Olá!

Meu nome é Rodrigo Morais um Software Enginner da Babbel e eu gostaria de compartilhar neste Blog com vocês minhas experiências e no que estou trabalhando.

Eu sou brasileiro, mas atualmento moro em Berlim. Tenho 19 anos trabalhando na área de desenvolvimento de software e sou apaixonado pelo que faço.

Comecei minha carreira com VB6 e SQL Server desenvolvendo um ERP para empresas de engenharia civil. Em 2003 quando .NET foi lançado e depois de alguma experiencia com ASP eu comecei minha aventura com o desenvolvimento Web. Todo esse tempo no desenvolvimento Web eu tive a oportunidade de trabalhar com diferentes plataformas e linguagens como Java and Ruby.

Atualmente minha principal linguagem e ambiente é JavaScript qual eu tenho trabalhodo com back-end e front-end. Eu tenho trabalho com Node.js para desenvolver serviços utilizando Express e com differentes frameworks como Ember, Meteor, AngularJS, React and elm para desenvolver o lado front-end.

Ultimamente meus interesses são Functional Programming, Haskell, React and elm e eu vou falar muito sobre isso aqui.

Espero que você gostem das discussões que teremos aqui e que nós possamos compartilhar o nosso conhecimento neste canal.

Obrigado!
